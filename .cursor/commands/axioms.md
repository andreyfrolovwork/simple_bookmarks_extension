# Аксиомы фичи

Фичу описываешь набором аксиом (законов). Или проверяешь: соответствует ли код заданным аксиомам.

**Режим A (описание):** контекст → список аксиом. Текст минимум, смысл максимум. Одна строка = одна истина.

**Режим B (проверка):** аксиомы → сверка с кодом. Для каждой: ✅ / ❌ + место.

## Формат вывода (режим A)

```
- аксиома 1
- аксиома 2
```

Без заголовков, нумерации, разделов.

## Формат вывода (режим B)

```
- аксиома 1 — ✅
- аксиома 2 — ❌ src/file.ts:42
```
