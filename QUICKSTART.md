# 🚀 Быстрый старт

## Установка и запуск (3 шага)

### 1️⃣ Установите зависимости
```bash
cd vitejs-vite-jbzr2frp
npm install
```

### 2️⃣ Запустите в режиме разработки
```bash
npm run dev
```

Откройте http://localhost:5173 в браузере. Приложение будет работать с тестовыми данными.

### 3️⃣ Соберите расширение для Chrome
```bash
npm run build
```

Затем:
1. Откройте `chrome://extensions/`
2. Включите "Режим разработчика"
3. Нажмите "Загрузить распакованное расширение"
4. Выберите папку `dist/`

## Готово! 🎉

Расширение установлено и готово к использованию. Кликните на иконку расширения в панели Chrome.
